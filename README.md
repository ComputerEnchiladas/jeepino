# Get Started

## MAHRIO Server 

* ```git clone https://github.com/ComputerEnchiladas/jeepino.git```
* ```npm install```
* ```node server.js```

## UV4L Server

-link to RaspberryPi-

### Setup UV4L On Startup

* ```crontab -e```

## Mobile App

### Run in Browser

* ```cd mobile/```
* ```ionic serve```

### Run on Device

* ```ionic upload```
